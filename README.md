# RattleShedX Website

A modern, responsive holding page for a creative digital workshop business.

## Features

- Modern design with video background
- Responsive layout for all devices  
- Newsletter signup with PHP backend
- Services dropdown menu
- Exhibition ready for iPad and QR codes

## Quick Start

1. Download all files to a folder
2. Start local server: python -m http.server 8000
3. Open http://localhost:8000

## File Structure

- index.html - Main website
- styles.css - All styling
- script.js - JavaScript functionality  
- subscribe.php - Email handling
- LICENSE - License file
- README.md - This file

## Setup

1. Update email in subscribe.php (line 6)
2. Replace the video URL in index.html
3. Upload to your cPanel hosting
4. Test on desktop and mobile

## Customisation

- Change colors in styles.css
- Update text in index.html
- Modify services in the dropdown menu
- Add your own video file
- Add your environment variables

## Support

Ensure your hosting supports PHP mail() function. Test the newsletter form after deployment.
